# skk-dict-imascgss
アイドルマスターシンデレラガールズ スターライトステージ SKK辞書

(現時点では)デレステに登場するアイドルたちの名前を一発で変換できるだけのSKK辞書です。  
作者が個人的に使えればいいので

* ソートしていません。AquaSKKで動作確認しています。
* (作者が本家シンデレラガールズをプレーしていないので)スターライトステージに実装されていないアイドルの名前は収録されていません。

ので、あらかじめご了承ください。
